# 演出の言葉

どうも、演出です。

突然ですが、僕は演劇が好きです。

今回、演出という仕事をさせて頂いて、好きな演劇と、演劇を作るということの間の大きなギャップに悩まされました。

何処か一人で悩んでしまう癖がある僕ですが、この劇を通じて人に頼ることを知る事ができた気がします。

それではご観劇ください。
